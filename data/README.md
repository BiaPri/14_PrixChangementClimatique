# Architecture

Ce dossier est destiné à l'équipe data : pour explorer la donnée disponible, ainsi que consolider la base de donnée partagée, qui sera notamment utilisée par l'équipe web.

Il est divisé comme suit :

## Dossier /exploration

Contient tout ce qui relève de l'exploration de donnée.
Import de la donnée, différents notebooks etc
Voir le readme : ./exploration/README.md

## Dossier /dbt_pipeline

C'est un projet dbt, qui ingère, structure et croise les données utilisées dans le cadre du projet.
Il contient génère la base de donnée pour le projet web.
Voir le readme : ./dbt_pipeline/README.md
