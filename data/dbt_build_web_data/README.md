### Run the projet

- uv run dbt seed
- uv run dbt run

Then open the output with duckdb (`duckdb --ui dev.duckdb`, see ../README.md for duckdb cli installation)

### Tests

- un run dbt test

### Open the doc

- uv run dbt docs generate
- uv run dbt docs serve
